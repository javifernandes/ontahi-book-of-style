# App Icons and Favicons

The ceibo symbol should work at multiple scales:

- app icon
- favicon
- GitHub avatar
- CLI badge
- PWA icon
- VSCode extension icon
- small documentation marker

Useful variants:

- full color on Forest
- full color on Paper
- negative on Ceibo
- monochrome Forest
- line variant for small technical contexts

At tiny sizes, preserve recognition over detail.
