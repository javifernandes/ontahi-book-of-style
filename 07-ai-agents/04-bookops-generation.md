# BookOps Generation Prompt

Use this prompt when asking an agent to generate BookOps material for Ontahí:

> Write a compact BookOps chapter for Ontahí using the Ontahí Language Kit. Keep the chapter editorial, precise and technically grounded. Prefer short sections, meaningful headings, restrained botanical references, clear diagrams or asset placeholders when useful, and language centered on executable ontology, entities, events, operations, tasks and durable operations. Do not treat workflows as the current core model. Avoid marketing copy and avoid expanding beyond what the implementation supports.
