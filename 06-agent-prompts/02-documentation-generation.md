# Documentation Generation Prompt

Use this prompt when asking an agent to write Ontahí documentation:

> Write Ontahí documentation in a precise, calm and conceptual tone. Explain ideas before mechanisms. Avoid hype. Prefer language such as executable ontology, living domain, entities with identity, events as facts, and tasks as durable operations. Use short sections, strong hierarchy and occasional literary subtlety. Do not overuse direct address.
