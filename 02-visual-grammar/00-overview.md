# Visual Grammar

The visual language of Ontahí is editorial and botanical.

It should resemble:

- a page from a carefully edited book
- a botanical atlas
- a scientific notebook
- a quiet museum label
- a warm code editor

It should not resemble:

- a SaaS landing page template
- a glowing AI dashboard
- a crypto protocol
- an enterprise admin panel

This is a grammar, not a gallery.

Rules should explain how small elements combine into a larger world.
