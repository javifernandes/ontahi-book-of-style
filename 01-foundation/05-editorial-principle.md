# The Editorial Principle

Ontahí should read like software made by people who care about books.

A page should have rhythm.

A diagram should have silence.

A code block should feel placed, not dumped.

BookOps is a natural editor for this language because it can combine text, diagrams, assets, snippets and examples inside one composed surface.

Documentation is literature for future engineers.
