# The Editorial Principle

\concept{Ontahí} should read like software made by people who care about books.

A page should have \term{rhythm}.

A diagram should have \term{silence}.

A code block should feel placed, not dumped.

\concept{BookOps} is a natural editor for this language because it can combine text, diagrams, assets, snippets and examples inside one composed surface.

\keyword{Documentation} is literature for future engineers.
