# Components

\concept{Components} should feel like \term{pieces of paper}, not floating glass.

## Cards

- warm background
- soft radius
- thin border
- minimal shadow
- generous padding

## Buttons

- calm
- clear
- not glossy
- not overly animated

## Panels

Use \term{dark panels} primarily for code editors, terminals and technical examples.
