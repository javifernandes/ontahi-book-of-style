# Typography

Typography carries the same responsibility as code: it gives structure to thought.

## Direction

- elegant serif for large headings and wordmarks
- readable humanist sans-serif for body text
- monospaced type for code
- generous line-height
- clear hierarchy
- no aggressive futurism

## Avoid

- square techno fonts
- overly playful display fonts
- dense corporate typography
- all-caps overuse
